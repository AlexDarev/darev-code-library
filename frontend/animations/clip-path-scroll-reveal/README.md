### Top → Bottom (Default)

```javascript
clipPath: "polygon(0 0, 100% 0, 100% 0%, 0 0%)"
```

```javascript
clipPath: "polygon(0 100%, 100% 100%, 100% 100%, 0 100%)"
```

```javascript
clipPath: "polygon(0 0, 0 0, 0 100%, 0 100%)"
```
